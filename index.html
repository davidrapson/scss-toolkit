SCSS Toolkit v2.1.0

SCSS Toolkit is a simple starter toolkit for Sass (SCSS) projects, based on SMACSS.

SCSS Toolkit is not a pattern library or theme framework, it's just a set of sensible defaults for organising new projects. It contains some sample modules, but these are just a starting point. See the README for more information.

Things it does include: Normalize, HTML5 Boilerplate default styles, and a few helpful functions and mixins (including Sass IE for working with media queries.)

Want a demo? You're looking at it, SCSS Toolkit was used to build this page.
